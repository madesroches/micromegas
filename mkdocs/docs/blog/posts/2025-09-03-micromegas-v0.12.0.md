# Micromegas v0.12.0: Async Span Tracing

We've significantly enhanced async span tracing for Rust applications, making high-frequency observability more accessible than ever.

<!-- more -->

## What's New

- Comprehensive async instrumentation with new proc macros
- Enhanced API for tracking complex async workflows
- Complete Python + SQL documentation
- Security updates and Rust 2024 edition support

## Why It Matters

- 20ns overhead per event — industry-leading performance
- 100k events/second capability per process
- Orders of magnitude cost savings vs traditional platforms
- Unified logs, metrics, and traces in one queryable format

Perfect for teams who need deep observability without breaking the budget.

Try it: `cargo add micromegas` or `pip install micromegas`
