# Doubling Down on FlightSQL

With the latest release of Micromegas I'm doubling down on FlightSQL. Not only does it power the connection with Grafana, it will also be the protocol under the Python API.

<!-- more -->

The spec can be intimidating, but you'd be surprised how little of the whole thing you need to get started and reap benefits.

Working in Rust? Check out the [FlightSQL implementation](https://github.com/madesroches/micromegas).

Interested in a unified observability solution that's really scalable? Join the project on [GitHub](https://github.com/madesroches/micromegas).
