Authentication endpoints for analytics-web-srv

Implements OIDC authorization code flow with PKCE:

  • /auth/login - Initiate OIDC login
  • /auth/callback - Handle OIDC callback
  • /auth/refresh - Refresh tokens
  • /auth/logout - Clear session
  • /auth/me - Get current user info

Security: All JWT tokens are fully validated (signature + claims) at this tier using the micromegas-auth crate with JWKS caching. Invalid tokens are rejected before forwarding requests to FlightSQL.
