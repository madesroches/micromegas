C ABI for the Micromegas telemetry producer stack.

Exposes init/shutdown/log/metric/flush as flat extern "C" functions that any non-Rust process can load via dlopen (cdylib) or link at build time (staticlib).

Threading model: all functions are safe to call from any thread. The transport runs on its own OS thread inside the library; callers need no async runtime.

String cardinality contract: mm_metric_i / mm_metric_f intern each unique (name, unit) pair via Box::leak. Keep metric names low-cardinality and bounded — do NOT pass unbounded values (session IDs, asset names) as metric names.
