micromegas_otel_ingestion

Source
Expand description

Glue between OTLP request bytes and the micromegas ingestion service.

Stays free of HTTP framework types so the same path works under axum, integration tests, or anything else that hands us a buffer of bytes. Errors map onto the OTLP/HTTP response surface in the server crate.
