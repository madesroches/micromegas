Approach 2 (revised) — Inline dividers, auto widths, pinned-width persistence
Initial state: all column widths are auto-computed from visible content — no user preference stored.
Dragging a divider pins that column to the user-chosen width (amber divider = pinned).
Right-click any divider → menu to reset that column to auto, or reset all at once.
"Reset widths" button appears in the pagination bar as soon as any column is pinned.
