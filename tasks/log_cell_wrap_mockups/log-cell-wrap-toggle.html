Wrap text toggle for the notebook Log cell

Interactive prototype of tasks/log_cell_multiline_messages_plan.md. Toggling Wrap text below switches every column from single-line truncate to whitespace-pre-wrap, so rows grow to show the full msg value — stack traces, chained errors, long payloads — instead of clipping it. Data and chrome are illustrative; behavior mirrors the plan.

On — default

Rows grow to fit their content immediately, no click needed. No virtualizer to fight — the cell already renders plain, paginated DOM rows. Persisted per-cell as options.wrapText (defaults to true when unset).

Off — one click

Every column single-lines and clips with an ellipsis. Full value only reachable via hover tooltip. Available for anyone who prefers the old compact view.
