Error States Reference

1. Network Error (Banner)
Unable to connect to server
The analytics server is not responding. This may be due to network issues or server maintenance.
Error: ECONNREFUSED 127.0.0.1:8000
✕
2. Query Error (Banner with SQL details)
Query execution failed
The SQL query contains a syntax error and could not be executed.
Error at line 3, column 15: Expected 'FROM' but found 'FORM'
✕
3. Query Timeout (Warning Banner)
Query timed out
The query took too long to execute (exceeded 30 seconds). Try narrowing your time range or adding more specific filters.
4. Full Page Network Error (when page cannot load at all)
Connection Failed
Unable to reach the analytics server. Please check your network connection and try again.
5. Empty State (No data for time range)
No processes found
No processes were found matching your criteria in the selected time range.
Suggestions:
  • Expand the time range to include more data
  • Check your search filters
  • Verify the application is sending telemetry
6. Empty State (No logs for process)
No log entries
This process has no log entries in the selected time range, or the log level filter is too restrictive.
Try:
  • Lowering the max log level to TRACE
  • Expanding the time range
  • Verifying log instrumentation is enabled
7. SQL Error (in SQL Panel)
SELECT process_id, exe FORM processes WHERE exe LIKE '%game%'
Syntax Error
Expected keyword 'FROM' but found 'FORM'
Line 2, column 1
8. Destructive Function Blocked
Query blocked for security
The query contains a destructive function that is not allowed in the web interface. Data modification operations require admin CLI access.
Blocked function: retire_partitions()
✕
9. Toast Notification (non-blocking errors)
Refresh failed
Connection refused (ECONNREFUSED 127.0.0.1:8000)
Retrying in 5s... Retry now
✕
