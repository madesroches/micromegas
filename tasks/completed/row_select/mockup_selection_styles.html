Row Selection in Table Cells

Issue #915 — Visual style options for selected rows. Click a row to select it; downstream cells reference it via $cell.selected.column.

Subtle left border indicator (3px blue) with faint background tint. Minimal — doesn't compete with hover state. Similar to VS Code's active line gutter.
